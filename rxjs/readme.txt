install lite-sever via npm en local
https://www.npmjs.com/package/lite-server

1. créer un fichier package.json vide avec juste les accolades
2. npm install lite-server --save-dev
3. vérifier si package.json ok avec dépendence lite-server
4. ajouter le code suivant dans package.json : 
   "scripts": {    
      "dev": "lite-server"
   }

lancer le server avec npm run dev
